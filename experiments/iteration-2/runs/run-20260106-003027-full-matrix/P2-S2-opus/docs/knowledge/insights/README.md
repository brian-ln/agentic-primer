# Insights

Observations and learnings from @copilot-assisted development sessions.

## What is an Insight?

An insight is a learning or observation that may be useful for future work:

- Best practices discovered
- Things that worked well
- Things that did not work
- Surprising behaviors
- Tips and tricks

## Insight Template

```markdown
# Insight: Title

**Date:** YYYY-MM-DD
**Source:** Issue #N / PR #N

## Observation
What was observed?

## Context
What were the circumstances?

## Takeaway
What should we remember for next time?

## Tags
#tag1 #tag2
```

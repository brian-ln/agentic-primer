## What Goes Here?

This directory captures:

- Agent-specific behaviors and quirks
- Performance observations
- Common failure patterns and solutions
- Tips for effective prompting
- Edge cases encountered

## Insight Template

```markdown
# Insight Title

## Summary
One-sentence description.

## Observation
What was noticed?

## Context
When/where was this observed?

## Implications
What does this mean for future work?

## Recommendations
Actionable suggestions.

## Evidence
Links to execution logs, issues, or PRs.
```

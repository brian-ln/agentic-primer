## Adding New Insights

1. Create insight file: `/docs/knowledge/insights/YYYY-MM-DD-title.md`
2. Add entry to this index under appropriate category
3. Create new category if needed
4. Format: `- [YYYY-MM-DD: Title](./YYYY-MM-DD-title.md) - One-line summary`

## Example Entry Format

```markdown
### Category Name
- [2026-01-06: Cache invalidation gotcha](./2026-01-06-cache-invalidation.md) - Cache keys must include version
- [2026-01-05: Batch API performance](./2026-01-05-batch-api.md) - Batching reduced calls by 90%
```

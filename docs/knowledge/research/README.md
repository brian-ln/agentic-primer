# Research Documents

This directory contains research findings and investigations.

## Document Structure

Each research document follows this template:

```markdown
# Research: [Title]

**Issue:** #XX
**Date:** YYYY-MM-DD
**Status:** In Progress/Complete

## Research Question
## Context
## Scope
## Methodology
## Findings
## Recommendations
## References
## Next Steps
```

## Automated Creation

Research documents are automatically created when issues with the `research` label are opened.

## Manual Creation

You can also create research documents manually by following the template above.

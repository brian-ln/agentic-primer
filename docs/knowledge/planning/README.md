# Planning Documents

This directory contains architecture documents and implementation plans.

## Document Structure

Each planning document follows this template:

```markdown
# Planning: [Title]

**Issue:** #XX
**Date:** YYYY-MM-DD
**Status:** In Progress/Complete

## Objective
## Requirements
## Context
## Architecture Overview
## Implementation Plan
## Task Breakdown
## Timeline
## Risk Assessment
## Success Criteria
## Next Steps
```

## Automated Creation

Planning documents are automatically created when issues with the `planning` label are opened.

## Manual Creation

You can also create planning documents manually by following the template above.
